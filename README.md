## Age Calculator App

This repository now includes a simple Windows WPF application that lets you select a birthday via a calendar and displays the age in years, months, and days with a glass-like interface.

To build and run the app (requires .NET 7 SDK on Windows):

```bash
dotnet build AgeCalculator
cd AgeCalculator
dotnet run
```
